# Impacta RH - Front-end

Front-end do sistema **Impacta RH**, um portal administrativo para gerenciamento de funcionários, desenvolvido como projeto acadêmico. A aplicação consome uma API em **ASP.NET Core** para realizar operações de cadastro, listagem, edição e exclusão de funcionários.

## Visão geral

O sistema foi construído com foco em uma interface administrativa moderna, com identidade visual inspirada na Faculdade Impacta. O front-end permite:

- visualizar indicadores na dashboard
- listar funcionários
- buscar por nome ou e-mail
- filtrar por cargo
- cadastrar funcionários
- editar funcionários
- excluir funcionários

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- Framer Motion
- Lucide React
